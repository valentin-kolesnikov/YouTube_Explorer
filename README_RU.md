## <p align="center"><a href="README.md">English</a> | Русский</p>

**YouTube Explorer** — это консольный инструмент на Python для программного исследования контента YouTube с использованием **YouTube Data API v3** (с поддержкой OAuth 2.0), **Return YouTube Dislike API** и **YouTube Transcript API**.

Он предназначен для:

- Работы в рамках лимитов квоты YouTube API  
- Получения и фильтрации комментариев  
- Поиска видео и получения плейлистов  
- Извлечения метаданных  
- Сбора субтитров

# ❗ Требования  

Для использования YouTube Explorer вам потребуется:

1. Желание использовать **YouTube Explorer**  
2. Действующий ключ YouTube Data API v3, полученный через Google Cloud Console  
3. Python 3.10 или новее (при запуске из репозитория)  
4. Стабильное интернет-соединение  

Рекомендуется базовое знание работы с командной строкой.

# ❓ Как получить ключ YouTube Data API v3?

1. Перейдите в [Google Cloud Console](https://console.cloud.google.com)  
2. Зарегистрируйтесь или войдите в аккаунт Google  
3. Нажмите `Create or select a project` → `New project`  
4. Укажите имя проекта (Google может задать его автоматически). Если у вас нет организации — не изменяйте поле `Location`. Нажмите `Create`  
5. Нажмите `Select project`, выберите проект. В поиске введите **YouTube Data API v3** → `Enable`  
6. В разделе API откройте вкладку `Credentials`  
7. Нажмите `Create credentials` → `API key`  
8. Скопируйте созданный ключ и сохраните его  

# Основные возможности

## 🔑 Обработка ключа YouTube API

- Первичная проверка введенного API-ключа перед запуском  
- Автоматическое создание и сохранение файла `Key.bin`  
- Отсутствие необходимости повторного ввода ключа после успешной проверки  

## 🚪 Проверка квоты

- Определение доступности текущей квоты YouTube API  
- Остановка выполнения, если дальнейшие запросы невозможны  

## 🛡️ Интеграция OAuth 2.0

- Приоритетное использование **OAuth 2.0**  
- Автоматическое переключение на метод **YouTube Data API Key**, если OAuth-данные отсутствуют  

## 💌 Comment Explorer

- Получение списка комментариев в консоли  
- Фильтрация по ключевым словам  
- Сортировка `по времени` или `по релевантности`  
- Ограничение количества выводимых результатов  
- Определение канала, на котором размещено видео  
- Вывод причины ошибки при её возникновении  

## 📹 Video Explorer

- Поиск видео по запросу  
- Поиск с учетом региона  
- Фильтрация по `диапазону дат публикации` и `продолжительности видео`  
- Ограничение количества результатов  
- Для каждого видео выводится:  
   1. Название  
   2. Прямая ссылка  
   3. Количество просмотров  
   4. Количество лайков  
   5. Количество дизлайков  
   6. Количество комментариев  
   7. Дата публикации  
   8. Название канала и ссылка на канал  

## 📈 Channel Explorer

- Сбор основной статистики канала:  
  - количество подписчиков  
  - количество просмотров  
  - описание  
  - дата регистрации  
  - ID канала (UC...)  
  - Хэндл (@...)  
- Возможность поиска видео внутри канала  

## 📄 Subtitles Explorer

- Ввод двухбуквенного кода языка (`en`, `ru` и т.д.)  
- Выбор между `созданными вручную` и `автоматическими` субтитрами  
- Автоматическое предложение альтернативы при отсутствии выбранного типа  

## 🔢 Info Explorer

- Ввод URL видео для прямого получения метаданных  
- Возврат полного блока информации из Video Explorer + описание видео  

# ⚙️ Функциональность

- **Аутентификация** — приоритет OAuth 2.0, при отсутствии используется ключ из `Key.bin`  
- **Парсинг ссылок** — автоматическое извлечение `video id`, `channel id`, `handle`  
- **Фильтры поиска** — `Region`, `Dimension`, `Duration`, `Date`  
- **Поиск по каналу** — поиск по ключевым словам внутри канала  
- **Умные субтитры** — интеграция YouTube Transcript API  
- **Дизлайки** — интеграция Return YouTube Dislike API  
- **Фильтрация комментариев** — локальная фильтрация по ключевым словам  
- **Проверка квоты** — тестовый запрос при запуске
