## [English](README.md) | Русский

**YouTube Explorer** — это консольный инструмент на Python для программного исследования контента YouTube с использованием **YouTube Data API v3** (с поддержкой OAuth 2.0), **Return YouTube Dislike API** и **YouTube Transcript API**

> [!CAUTION]
> Официальный репозиторий доступен только на моем [GitHub](https://github.com/valentin-kolesnikov/YouTube_Explorer).
> Любые клоны, распространяемые от моего имени на других платформах, не являются моей работой.

> [!WARNING]
> Некоторое антивирусное программное обеспечение может помечать приложение как потенциально нежелательную программу (PUP). Это ложное срабатывание антивируса.
> Все материалы, используемые для приложения, хранятся в публичном репозитории.  
> У меня как разработчика нет желания вредить другим пользователям. Вы всегда можете обратиться ко мне через email по таким вопросам.

YouTube Explorer предназначен для:

- Работы в пределах лимитов квоты YouTube API
- Получения и фильтрации комментариев
- Поиска видео и получения плейлистов
- Извлечения метаданных
- Сбора субтитров

# ❗Требования  

Для использования YouTube Explorer вам понадобится следующее:

1. Ваше желание использовать **YouTube Explorer**
2. Действительный ключ YouTube Data API v3, выпущенный через Google Cloud Console
3. Python 3.10 или новее (требуется при запуске из репозитория)
4. Стабильное интернет-соединение

Рекомендуется базовое знакомство с использованием командной строки.

# 🌐Как скачать?

## Releases

Windows:
1. `Releases` → `YouTubeExplorer.zip`
2. Извлеките папку `.zip`
3. `bin` → откройте `YouTubeExplorer.exe`

Ubuntu-подобные:
1. **Releases** → `YouTubeExplorer.tar.gz`
2. Извлеките папку `.tar.gz`
3. YouTubeExplorer.dist → Скопируйте `YouTubeExplorer.bin` → Вставьте в терминал и нажмите Enter, или же запустите командой: `./YouTubeExplorer.bin`

## Git

1. Установите `Git`
2. В терминале вставьте: `git clone https://github.com/valentin-kolesnikov/YouTube_Explorer`
3. Перейдите в папку и запустите основной скрипт `YouTubeExplorer.py`
4. Введите `pip install -r requirements.txt` в терминале, чтобы установить все зависимости программы

# ❓Как получить ключ YouTube Data API v3?

1. Перейдите в [Google Cloud Console](https://console.cloud.google.com)

2. Вам нужно зарегистрировать аккаунт Google или войти в него.

3. Нажмите кнопку **Create or select a project** в центре страницы → **New project**.

4. Введите название проекта (или оставьте сгенерированное автоматически). → Если вы не принадлежите к какой-либо организации, оставьте **Location** по умолчанию. → Нажмите **Create**.

5. На этой же странице нажмите **Select project** и выберите свой проект. Введите `YouTube Data API v3` в строке поиска → нажмите **Enable**.

6. Вас перенаправит на страницу конфигурации API. В левом столбце нажмите **Credentials**.

7. Вверху нажмите **Create credentials** → **Ключ API**.

8. Скопируйте ваш API-ключ и вставьте его в YouTube Explorer. При необходимости вы можете восстановить ключ из главного меню.

# 🛡️ Как получить учетные данные OAuth 2.0?

- Приложение разработано для использования **OAuth 2.0** в качестве основного метода аутентификации
- Автоматическое переключение на метод **YouTube Data API Key**, если OAuth-данные отсутствуют 

Поскольку автоматическая регистрация OAuth 2.0 не поддерживается из коробки, потребуется сгенерировать собственные учетные данные, выполнив следующие шаги:

1. Перейдите в [Google Cloud Console](https://console.cloud.google.com).

2. Выберите проект, который вы уже создали для своего ключа YouTube Data API v3.

3. В левом столбце нажмите **OAuth consent screen**.

4. Выберите **External** в качестве типа пользователя и нажмите **Create**.

5. Заполните обязательные поля: `App name`, `User support email поддержки` и `Developer contact information` (можно использовать свою собственную электронную почту). Нажмите **Save and Continue**.

6. Пропустите страницу **Scopes**, нажав **Save and Continue**.

7. На странице **Test users** нажмите **Add Users** и введите адрес электронной почты своего аккаунта Google. Нажмите **Save and Continue**.

8. Теперь снова нажмите **Credentials** в левом столбце.

9. Вверху нажмите **Create credentials** → **OAuth client ID**.

10. Выберите **Desktop app** в качестве типа приложения. Вы можете оставить имя по умолчанию. Нажмите **Create**.

11. Появится всплывающее окно. Нажмите **Download JSON**, чтобы сохранить ваши учетные данные.

12. Скачанный файл `.json` необходимо поместить в папку `Keys`, которая находится в папке `bin`.

# Основные возможности

## 🔑Работа с ключом YouTube API

- Первоначальная проверка предоставленного API-ключа перед выполнением
- Автоматическое создание и сохранение файла `Key.bin`
- Исключение повторного ввода ключа после успешной проверки

## 🚪Проверка квоты

- Определяет текущую доступность квоты YouTube API
- Останавливает выполнение, если дальнейшие запросы невозможны

## 💌Comment Explorer

- Получайте список комментариев к видео на YouTube в консоли
- Фильтруйте комментарии по ключевому слову
- Сортируйте `по времени` (by time) или `по релевантности` (by relevance)
- Ограничивайте количество выводимых результатов
- Определяйте канал, на котором размещено видео
- НОВЕЙШЕЕ! Comment Explorer может сохранять собранные данные в файл `.docx`

## 📹Video Explorer

- Ищите видео по вашему запросу
- Выполняйте поиск с учетом региона
- Применяйте дополнительные фильтры, такие как: `диапазон дат публикации` и `длительность видео`
- Контролируйте максимальное количество возвращаемых результатов поиска
- Для каждого видео вы получите следующие метаданные:
   1. Название видео
   2. Прямая ссылка
   3. Количество просмотров
   4. Количество лайков
   5. Количество дизлайков
   6. Количество комментариев
   7. Дата публикации
   8. Название канала и ссылка на канал

## 📈Channel Explorer

- Собирайте большую часть статистики канала:
  - Количество подписчиков  
  - Количество просмотров  
  - Описание  
  - Дата регистрации
  - ID канала (UC...)
  - Имя пользователя (Handle) (@...)
- При необходимости у вас есть возможность искать видео на опрделённом канале благодаря компонентам Video Explorer

## 🗂️Playlist Explorer

- Ищите плейлисты с помощью ключевых слов или безопасно загружайте **свои собственные** плейлисты, используя OAuth 2.0
- Просматривайте основные сведения о плейлисте, такие как статус конфиденциальности, общее количество видео и дата публикации
- Легко извлекайте все видео из любого плейлиста, просто введя его URL (поддерживаются ссылки как `PL...`, так и `OL...`)
- Получайте подробную статистику для каждого видео внутри плейлиста, включая количество дизлайков
- Анализируйте видеотеку плейлиста сразу после его нахождения, без необходимости возвращаться в главное меню

## 📄Subtitles Explorer

**Subtitles Explorer** может извлечь субтитры из видео без всяких усилий

- Введите двухбуквенный код языка (`en`, `ru` и т.д.)  
- Выберите между `созданными вручную` для большей точности и `автоматическими` субтитрами, если субтитров, сделанных вручную, не окажется  
- Автоматическое предложение альтернативы при отсутствии выбранного типа  
- Если тип расшифровки недоступен, программа не выдаст ошибку и не прекратит работу. Вместо этого она обнаружит проблему и сразу предложит альтернативную версию, чтобы вы смогли получить желаемое.

## 🔢Info Explorer

- Введите URL-адрес конкретного видео, чтобы напрямую извлечь метаданные без поиска
- Возвращает полный блок информации, описанный в [Video Explorer](#video-explorer), а также описание видео

# ⚙️ Функционал

- **Аутентификация** — **OAuth 2.0** имеет приоритет для безопасного доступа. Если учетные данные OAuth не обнаружены, приложение использует **YouTube Data API key**, сохраненный в `Key.bin`
- **Парсинг ссылок** — `video id`, `channel id` (UC...) и `имя пользователя` (@...) автоматически извлекаются из введенных вами ссылок
- **Фильтры поиска** — вы можете фильтровать видео по `Региону` (Region), `Измерению` (Dimension) (2D или 3D), `Длительности` (Duration) и `Дате` (Date)
- **Поиск по каналу** — в отличие от стандартного поиска, вы можете выполнять поиск по ключевым словам непосредственно внутри библиотеки канала
- **Умные субтитры** — интегрирован **YouTube Transcript API**. Сначала он ищет `созданные вручную` (manually created) субтитры. Если они отсутствуют, он спросит, хотите ли вы получить `автоматически сгенерированные` (generated)
- **Дизлайки** — интегрирован **Return YouTube Dislike API** для отображения количества дизлайков в сочетании с официальными данными
- **Фильтрация комментариев** — **Ключевые слова** используются для локальной фильтрации комментариев и ответов. YouTube Data API не осуществляет такую фильтрацию
- **Проверка квоты** — при запуске делается тестовый запрос, чтобы убедиться, что ваша **квота YouTube API** не превышена до начала работы
- **Автоматическая проверка** — перед выполнением запросов к API, YouTube Explorer пингует `google.com`, чтобы убедиться, что ваше соединение активно. Если соединение разорвано, вы можете повторить попытку подключения
- Получайте подробные сообщения об ошибках, если что-то пойдет не так
- Главное меню состоит из двух страниц
- Восстановите свой ключ **YouTube API** на второй странице главного меню
